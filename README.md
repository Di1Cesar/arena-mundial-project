# Arena Mundial

Projeto React + Vite inspirado na identidade visual enviada.

## Stack
- React
- Vite
- TailwindCSS
- React Router
- Estrutura preparada para shadcn-ui

## Instalação

```bash
npm install
npm run dev
```

## Estrutura
- Sidebar fixa
- Dashboard
- Classificação
- Partidas
- Times
- Rotas configuradas
